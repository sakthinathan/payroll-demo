# ☁️ Skyline Payroll — Enterprise Demo

Welcome to the **Skyline Payroll** demonstration environment. This is a white-labeled version of our enterprise-grade payroll management platform, designed specifically for high-growth agencies and service-based businesses.

## 🚀 Key Features

- **Multi-Cycle Processing**: Effortlessly manage both Weekly and Monthly payroll cycles.
- **Automated PDF Reports**: Generate professional, audit-ready payroll reports with one click.
- **Bank Integration**: Export bulk-payment files compatible with major banks (e.g., SBI).
- **White-Label Ready**: Dynamic branding and color systems for client-specific skinning.
- **Privacy-First**: This demo environment is populated with 100% anonymized employee data.

## 🛠️ Technology Stack

- **Frontend**: React.js with Vite for high-performance rendering.
- **Styling**: Vanilla CSS with a responsive grid system for cross-device support.
- **Database & Auth**: Supabase (PostgreSQL) for real-time data persistence.
- **Animations**: Framer Motion for a premium, smooth user experience.
